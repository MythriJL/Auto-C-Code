# Auto-C-Code
 
 
# this Project converts the psedocode to c code


# Requirement

i) gcc
ii) lex
iii) yacc

# steps to run the code:

i) clone the repo
ii) change the present working directory to the clonned folder
iii) execute the following commands
      a) lex lexfile.l
      b) yacc -d yaccfile.y
      c) cc lex.yy.c y.tab.c -ll
      d) ./a.out
